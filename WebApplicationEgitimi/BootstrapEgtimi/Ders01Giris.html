Bootstrap Eğitimi

Bootstrap responsive(mobil uyumlu) web siteleri yapabilmemiz için geliştirilmiş, içerisinde css ve javascript kütüphanesi olan bir freameworktür.

Bootstrapi Projeye Yükleme

Mvc projelerinde bootstrap de yüklü gelir. Web form projesinde ise biz yükleriz.

Yükleme Konumları

  • Bootstrap web sitesinden
  • Online CDN Kütüphane sitelerinden
  • Nuget paket yöneticisinden

Bootstrap varsayılan olarak ekranı dikeyde excel dekine benzer şekilde 12 parçaya bölerek responsive tasarım uygular !!

Container

Bs de container classı ile içerikler sarmalanır ve bu şekilde site responsive hale getirilmiş olur.

Container Kullanım Örneği

bir dive container class ı vererek basit bir şekilde responsive hale getirebiliriz
Container classı maksimum ölçüye geldiğinde kenarlardan boşluk bırakarak divi ortalar
Bir class="container-fluid" kodu ile class eklersek ekran genişliğinin tamamını kullanarak responsive tasarım kullanabiliriz.
