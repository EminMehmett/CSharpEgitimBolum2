Bootstrapde Margin ve Padding kullanımı

Margin Kullanarak Hizalama

Kolon 1 - col-md-4
Kolon 2 - col-md-4 ms-auto // ms-auto ile divi sağa hizalar
Kolon 1 - col-md-3
Kolon 2 - col-md-3 me-auto // me-auto ile divi sola hizalar

Bootstrap İç-Dış Boşluk Kullanımı

Kolon 1 p-5
Kolon 2 p-3
Kolon 3 p-1

Bs ta elementlere p-0 ile p-5 classlarıyla padding verilebilir.

p-sm, md,lg .. kullanarak farklı ekran genişlikleri için farklı boşluklar da kullanabiliriz.

Kolon 1 m-5
Kolon 2 m-3
Kolon 3 m-1

Bs ta elementlere m-0 ile m-5 classlarıyla margin verebiliriz

m-sm, md, lg.. kullanarak farklı ekran genişlikleri için farklı boşluklar da kullanabiliriz
ms-5 =baştan yani sola margin, me-2 =margin right, pt-3 = padding-top, pb-4 = padding-bottom kodlarıyla tek yönden margin veya padding uygulayabiliriz.

Bootstrap Metin Hizalama

Bootstrap ta element içerisindeki metinleri sağa, sola veya merkeze class kullanarak hizalayabiliriz.

Yazıyı ortalamak için class="text-center" classı kullanılır.
Yazıyı sağa hizalamak için class="text-end" classı kullanılır.

Bootstrap Responsive Images

Bootstrap te resimleri de mobil uyumlu hale getirmek için class kullanabilmekteyiz.

Kullanımı

Resme class="img-fluid" classı vererek resmi responsive hale getirdik
Resme class="img-thumbnail" classı vererek resmi responsive hale getirdik
