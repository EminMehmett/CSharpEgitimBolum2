Css de arka plan rengine opacity ile saydamlık, şeffaflık ekleyebiliriz
Gerekli kod : background-color: green; opacity: 0.5;
Opaklık için bir diğer yöntem de style da renk kodu olarak rgba kullanmaktır
Örnek : background: rgba(0, 128, 0, 0.5)
Sondaki değer alfa dır opaklığı bu değer ayarlar

Css arka plan resmi

CSs ile html elemanlarına arka plan resmi ekleyebiliriz

Bunun için yazmamız gereken kod şudur : background-image: url(arkaplan.gif)'

Css arka plan tekrarı

CSs ile html elemanlarına arka plan tekrarı ekleyebiliriz

background-repeat özelliği ile resim tekrarını yatayda, dikeyde veya tekrar etmeyecek şekilde ayarlayabiliriz

  1. background-repeat:repeat-x; : arka plan resmini yatayda tekrar et
  2. background-repeat:repeat-y; : arka plan resmini dikeyde tekrar et
  3. background-repeat:no-repeat; : arka plan resmini tekrar etme

Css arka plan konumu

CSs ile html elemanlarına arka plan konumu ekleyebiliriz

Bunun için yazmamız gereken kod şudur : background-image: url(arkaplan.gif)' Bütün uzantılar(png, jpg vb) olabilir

Arka plan resminin pozisyonunu ayarlamak için gerekli kod : background-position:right top; veya bacckground-position: 18px 58px; pixellerle de ayarlayabiliriz

Css arka plan sabitleme

Arka plan resminin kaydırılması mı yoksa sabit mi olması gerektiğini belirler

Eğer sayfa aşağı kaydırıldığında arka planresminin sabit kalmasını istersek şu kodu eklemeliyiz : backgroud-attachment: fixed;

"background: #fff url('/Images/logo.png') no-repeat fixed center bottom; height: 500px;" : kodu ile tek satırda tüm arka plan işlemlerini halledebiliriz
