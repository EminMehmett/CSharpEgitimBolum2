CSS Seçiciler

Css söz dizimi(Syntax)

Css kural kümesi bir seçici ve bildirim bloğundan oluşur:

Örneğin h1{color:blue;} şeklinde bir bildirimde bulunursak burada h1 seçici yani css i uygulamak istediğimiz elemandır, küme parantezi içindeki color property yani özelliktir, blue ise bu özelliğe atanan değer (value) dir

id seçici - Css de html elementlerine id attribute ü tanımlayarak o elementi seçebiliriz

Bu div elementini id özelliğine göre seçip Css uygulayabiliriz

class seçici - CSS de html elementlerine class attribute ü tanımlayarak o elementi seçebiliyoruz

Bu div elementini class özelliğine göre seçip Css uygulayabiliriz

CSS nasıl eklenir

CSS eklemenin üç yolu vardır

  1. Harici CSS
  2. Dahili CSS
  3. Satır içi (inline) CSS
Satır içi (inline) css

SAtır içi(inline) CSS: Bu elementte kullandığımız style="background-color:brown; color: bisque;" kodları ile yazım şekline inline CSS denir. Bu şekilde de html elementlerine css yazabiliriz

style kullanarak html kontrolüne yazı fontu verme

Html elementlerinde border ile çerçeve kullanabiliriz

Html elemetlerinde renk ismi yerine renk kodlarını da kullanabiliriz

Html öğesi için birden fazla stil belirtildiğinde hangi stil kullanılır?

1-Satır içi stili(inline CSS) (html öğesi içinde h3 style="color: blue;" şeklinde yazarsak)
2-Dış ve iç stil sayfaları (baş bölümde belirttiğim gibi css lerin sırasına göre)
3-Tarayıcı varsayılan(her tarayıcının html elemanlarına uyguladığı varsayılan css özellikleri vardır)
